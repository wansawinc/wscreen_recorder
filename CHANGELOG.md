## 1.0.0 
* 🛠 Flutter & Android Compatibility (Flutter 3.x / Gradle 8.14 / AGP 8.x)
* Removed hardcoded buildscript classpath for AGP.
* Updated compileSdkVersion and targetSdkVersion to 34.
* Upgraded Java compatibility to Java 17.
* Gradle Upgrade: Updated Gradle Wrapper to v8.14 and Android Gradle Plugin to v8.3.0 in both plugin and example app.
* Uuid Fix: Updated uuid constraint to '>=3.0.6 <5.0.0' to resolve conflicts with modern apps using uuid v4.
* Linting: Updated flutter_lints to v3.0.0.
* Android 14+ Support: Added FOREGROUND_SERVICE_MEDIA_PROJECTION permission to ensure screen recording works on Android 14 and newer.
